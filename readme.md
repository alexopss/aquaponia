Resumo

Resumo – Introdução: A aquaponia trata da combinação de dois conceitos consagrados, a Piscicultura (criação 
de peixes) e a Hidroponia(cultivo de plantas sem solo). Crescendo e se reproduzindo em conjunto plantas e 
peixes suprem a necessidade um do outro. A aquaponia é um sistema que simula o efeito natural do benefício 
decorrente desta da necessidade mútua. Os peixes produzem compostos em seus excrementos, como por 
exemplo Amônia (NH3), que em grandes quantidades são extremamente tóxicos aos peixes. Porém para os 
vegetais, estes componentes são nutrientes importantes para o desenvolvimento das plantas que, por sua vez, 
absorvem esses compostos, filtrando naturalmente a água, deixando-a limpa e apta para reutilização no 
reservatório de peixes. Metodologia: Este estudo visa implementar um sistema de monitoramento e controle 
para um sistema de aquaponia, utilizando-se da tecnologia de computação para adquirir, processar, analisar e 
gerar dados como subsídio para a otimização da produção, isto é, o melhor crescimento da cultura e da criação 
de peixes e de plantas. Serão utilizados dois reservatórios, um de vidro (aquário), um para acomodar os peixes 
e outro para acomodar as plantas. Um controlador de temperatura da água será desenvolvido para manter a 
temperatura dentro de uma faixa aceitável para o crescimento e reprodução dos peixes e das plantas. 
Desenvolvimento: O sistema foi implementado utilizando plataformas computacionais embarcadas do tipo 
Arduino (Arduino Mega 2560), componentes agregados como sensores, relés e componentes discretos para 
interação (interfaces). A integração da plataforma Arduino com um banco de dados MySQL (Structured Query 
Language), juntamente uma API-PHP (Application Programming Interface – Hypertext Preprocessor ) para 
comunicação, na atuação sobre o aquecimento e o resfriamento do aquário um emissor de calor simples, uma 
lâmpada incandescente, e um redutor de calor, um ventilador em miniatura (cooler) acoplados ao aquário. 
Conclusão: A implementação do sistema de monitoramento e controle de aquaponia poderão ser criados 
históricos, tabelas, gráficos de tendências que permitam o planejamento e a otimização, da produção de 
Sistemas de Aquaponia e de sistemas similares de produção.
